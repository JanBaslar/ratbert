# RATBERT 🐀

Macaroni and cheese causes paranoia.

## Prerequisites 🛠️

To run this project, you need to have the following installed:

* **Python 3.9+** ([Download Python](https://www.python.org/downloads/))
* **Git** ([Download Git](https://git-scm.com/downloads))

## How to Use 🚀

Follow these steps to get the project up and running on your local machine:

### 1. Clone the Repository
Open your terminal or command prompt and run:
```bash
git clone https://github.com/JanBaslar/ratbert.git
```

### 2. Go to the Directory
Run the following command to navigate into the project directory:
```bash
cd ratbert
```

### 3. Run the App
Run this command to run the application:
```bash
python ratbert.py
```
This will create a virtual environment (If run for the first time), install the required dependencies, and launch the application.


> ⚠️ Make sure to run the command from the root **ratbert** directory to ensure the application can find all necessary files.
